# 💰 Expense Tracker

A simple and responsive Expense Tracker built using **HTML, CSS, and JavaScript**. This application helps users manage their income and expenses while automatically calculating the current balance. All data is stored in the browser using **Local Storage**.

## 🚀 Live Demo

🔗 https://chitranshchauhan.github.io/Expense-Tracker/

## ✨ Features

- ➕ Add Income
- ➖ Add Expenses
- 💰 Automatic Balance Calculation
- 📋 Transaction History
- 🗑️ Delete Transactions
- 💾 Local Storage Support
- 📱 Responsive Design
- ⚡ Fast and Easy to Use

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## ⚙️ How to Use

1. Enter the transaction description.
2. Enter the amount.
3. Select **Income** or **Expense**.
4. Click **Add Transaction**.
5. View the updated balance and transaction history.
6. Delete any transaction using the **Delete** button.

## 💾 Data Storage

This project uses **Local Storage**, so your transactions remain saved even after refreshing the browser.
